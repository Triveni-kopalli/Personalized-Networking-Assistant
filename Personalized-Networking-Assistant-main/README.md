# 🤝 Personalized Networking Assistant

An AI-powered networking assistant that helps users build meaningful professional connections by providing intelligent recommendations, profile analysis, and networking support. This project combines Natural Language Processing (NLP), FastAPI, and Streamlit to deliver an interactive and user-friendly experience.

---

## 📌 Project Overview

The Personalized Networking Assistant is designed to simplify professional networking by leveraging Artificial Intelligence. It analyzes user queries and provides personalized recommendations, helping users improve communication, discover networking opportunities, and receive intelligent assistance.

- 🔍 Intelligent networking assistance

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Personalized-Networking-Assistant.git
```

```bash
cd Personalized-Networking-Assistant
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

Command Prompt

```bash
venv\Scripts\activate
```

PowerShell

```powershell
.\venv\Scripts\Activate.ps1
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Backend

```bash
uvicorn backend.main:app --reload
```

Backend runs at:

```
http://127.0.0.1:8000
```

---

## ▶️ Run the Frontend

Open another terminal.

```bash
streamlit run frontend/app.py
```

Application opens at

```
http://localhost:8501
```
